# failure log schema

## daily entry format

each daily entry follows this structure:

### header
```
# [date] - failure log

**day:** [day of week]
```

### sections

#### 1. failures/issues
what went wrong? when? impact?

#### 2. root cause
why? contributing factors? preventable?

#### 3. lessons
what'd you learn? insight?

#### 4. resolution & next steps
- how'd you fix it?
- action items:
  - [ ] 
  - [ ]

#### 5. reflection
how do you feel? what's different next time?
